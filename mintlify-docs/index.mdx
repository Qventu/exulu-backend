<AccordionGroup>
  <Accordion title="🤖 Multi-Agent Systems" defaultOpen>
    Create sophisticated AI agents powered by leading LLM providers. Build single agents or complex multi-agent workflows with built-in orchestration. Each agent can have different capabilities, tools, and providers.

    ```typescript
    import { ExuluAgent } from "@exulu/backend";
    import { createAnthropic } from "@ai-sdk/anthropic";

    const researchAgent = new ExuluAgent({
      id: "research_assistant",
      name: "Research Assistant",
      provider: "anthropic",
      description: "AI agent specialized in research tasks",
      type: "agent",
      capabilities: {
        text: true,
        images: [".png", ".jpg", ".jpeg", ".webp"],
        files: [".pdf", ".docx"],
        audio: [],
        video: []
      },
      maxContextLength: 200000,
      config: {
        name: "Research Assistant",
        instructions: "You are a helpful research assistant...",
        model: {
          create: ({ apiKey }) => {
            const anthropic = createAnthropic({ apiKey });
            return anthropic.languageModel("claude-sonnet-4-5");
          }
        }
      }
    });
    ```
  </Accordion>

  <Accordion title="🔍 Semantic Search & RAG">
    Production-ready semantic search with vector embeddings, hybrid search (vector + full-text), and flexible chunking strategies. Perfect for building RAG applications.

    ```typescript
    const context = new ExuluContext({
      id: "documentation",
      embedder: openAIEmbedder,
      fields: [
        { name: "title", type: "string" },
        { name: "content", type: "string" }
      ]
    });
    ```
  </Accordion>

  <Accordion title="📊 Dynamic GraphQL API">
    Your schema is automatically generated from your context definitions. Define your data structures once and get a complete CRUD API with filtering, pagination, and vector search - no manual schema writing.
  </Accordion>

  <Accordion title="🔄 Conversation Templates">
    Unlike traditional automation workflows, Exulu provides a unique system of conversation templates—predefined conversations where you define input messages, outputs, and subsequent interactions. Simply chat with an AI agent to design your workflow, then save it as a template. This intuitive approach lets you capture complex multi-step reasoning patterns and reuse them across your application. Schedule template conversations with cron expressions for automated execution.
  </Accordion>

  <Accordion title="✅ Evaluation & Testing">
    Built-in evaluation framework for testing agent performance. Create test cases, run evaluation sets, and measure quality with custom evaluation functions and LLM-based judges.
  </Accordion>

  <Accordion title="🔐 Role-Based Access Control">
    Fine-grained RBAC system with user and role-based permissions. Control access at the resource level with read/write permissions and private/public modes.
  </Accordion>

  <Accordion title="⚙️ Background Job Processing">
    Built on top of BullMQ, Exulu provides a full API and UI for managing embedding generation, chunk processing, and agent execution with configurable queues, rate limits, and retries.
  </Accordion>

  <Accordion title="📈 Observability & Monitoring">
    OpenTelemetry integration for comprehensive observability. Track agent performance, monitor API usage, and analyze system behavior with distributed tracing and metrics.
  </Accordion>
</AccordionGroup>
